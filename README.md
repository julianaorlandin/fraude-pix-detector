# Detector de Fraude em Transações PIX

Projeto em Python para simulação e detecção de fraude em transações PIX.

## Objetivo

Simular transações financeiras e aplicar regras de análise para identificar possíveis fraudes.

## Tecnologias

- Python
- Pandas
- Matplotlib

## Estrutura do projeto

src/
- gerador_transacoes.py
- detector_fraude.py

data/
- transacoes_pix.csv

## Como executar

Gerar dados simulados:

python src/gerador_transacoes.py

Analisar fraudes:

python src/detector_fraude.py
